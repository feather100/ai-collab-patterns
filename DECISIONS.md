## 1. 决策记录格式

每条决策记录（DR-xx）包含：

| 字段 | 说明 |
|---|---|
| **标题** | 改了什么（一句话） |
| **背景** | 为什么需要这个决策 |
| **决策** | 具体做了什么决定 |
| **备选方案（可选）** | 考虑了哪些其他方案，为什么没选 |
| **理由** | 为什么选这个方案 |
| **过程留痕（可选）** | 踩到的坑、过程中的发现 |
| **边界** | 这次决策不覆盖什么、什么条件下可能失效 |
