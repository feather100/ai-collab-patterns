# 示例参考

本目录包含 AI 协作协议模板的使用示例，**所有内容均为占位演示**，用于展示文件格式和用法，不涉及任何真实项目信息。

## 文件说明

| 文件 | 用途 |
|---|---|
| `AGENTS_EXAMPLE.md` | AI_ENTRY.md 的格式示例 |
| `DECISIONS_EXAMPLE.md` | 决策记录的格式示例 |
| `COLLAB_EXAMPLE.md` | 协作规范的格式示例 |
| `SESSION_HANDOFF_EXAMPLE.md` | 会话交接的格式示例 |

## 使用方式

参考这些示例的格式，将模板文件（`AI_ENTRY.md`、`DECISIONS.md`、`COLLAB.md`、`SESSION_HANDOFF.md`）中的内容替换为你的实际项目信息。
