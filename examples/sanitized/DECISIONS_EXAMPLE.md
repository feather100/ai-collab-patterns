## 1. 决策记录格式

每条决策记录包含：

| 字段 | 说明 |
|---|---|
| **标题** | 改了什么 |
| **背景** | 为什么需要这个决策 |
| **决策** | 具体决定 |
| **理由** | 为什么选这个方案 |
| **边界** | 不做什么 |
